## Conventions

- **TY (this year) primary:** `#C4956A` (index 3)
- **LY (last year) comparison:** `#c4956a47` (index 3 at 28% opacity)
- For period-over-period charts, use the solid color for TY and the `47` hex-alpha suffix for LY
- For multi-series charts (e.g., stacked bars), assign colors from the palette in index order starting from the most prominent series
